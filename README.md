# The Artifacts

### 📖 Description
장식된 도자기를 꾸미는 데만 사용하던 도자기 파편으로 유용한 유닛을 만들어 보세요.

> 삭제하기: `/function ce:uninstall {key:QUAM12}`

   - 코드명: `ceramics-unit`

# 유닛
유닛은 도자기로 구성된 개체의 가장 작은 단위입니다. 명령을 따라 공격, 행동하며 기본적으로 살육에 대한 거부감이 없습니다.  
거의 대부분의 장비를 착용할 수 있으며, 특정 블록/아이템/개체에 대하여 상호작용 합니다.

일정 시간이 지나면 자동으로 소멸됩니다. (세계가 초기화 되기 전, 환경 오염을 방지하기 위한 목적이라는 사료가 존재합니다)
**어떤 재료**는 소멸 시간을 늘이거나, **소멸 되지 않게**할 수 있습니다.

# 소환
레시피를 따라 유닛을 제작한 후, 바닥 위에 우클릭 하세요. 유닛이 소환됩니다.
- 바닥이 존재하지 않으면 바닥이 있는 곳으로 떨어지거나, 공허로 추락합니다.
- `무중력` 기능을 활성화 하면 떨어지지 않습니다.
